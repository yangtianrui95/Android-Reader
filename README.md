# Android-Reader
A Android app to search book in Douban.
一个简单的书籍查询客户端，使用豆瓣API


![Reader介绍](http://img.blog.csdn.net/20160601143703010)

##使用技术:
1. Metrial Design设计
2. okhttp，gson，agera处理网络请求
3. glide加载图片
